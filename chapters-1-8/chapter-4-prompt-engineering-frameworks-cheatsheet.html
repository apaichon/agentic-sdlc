FRAMEWORK DETAILS & EXAMPLES

1RTF

Structure:

  • Role
  • Task
  • Format

Example: Act as an English tutor. Explain active vs passive voice as a table.

2TAG

Structure:

  • Task
  • Action
  • Goal

Example: Task: Summarize. Action: Highlight 5 points. Goal: Understand quickly.

3CO-STAR

Structure:

  • Context
  • Objective
  • Style
  • Tone
  • Audience
  • Response

4CLEAR

Structure:

  • Context
  • Length
  • Examples
  • Audience
  • Role

5RACE

Structure:

  • Role
  • Action
  • Context
  • Expectation

Example: You are a history tutor. Explain causes of World War I.

6RISEN

Structure:

  • Role
  • Instructions
  • Steps
  • End Goal
  • Narrowing

7SCOPE

Structure:

  • Situation
  • Constraints
  • Objective
  • Persona
  • Expected Output

8APE

Structure:

  • Action
  • Purpose
  • Expectation

Example: Create a revision checklist for a math exam.

9CARE

Structure:

  • Context
  • Action
  • Result
  • Example

Example: Proofread this paragraph and explain fixes.

10FEW-SHOT

Structure:

  • Example input
  • Example output
  • New task

Use: Style matching and classification.

11ZERO-SHOT

Structure:

  • Ask directly without examples

Example: Explain gravity using one simple example.

12ReAct

Structure:

  • Reasoning
  • Acting

Use: Research, tools, and problem solving.

13REFLEXION

Structure: Draft → Critique → Improve

Example: Write a paragraph, critique it, then rewrite it better.

14RAG

Structure: Use provided documents as context.

Example: Answer only from the notes below. Do not add outside information.

TIPS FOR BETTER PROMPTS

  • Be clear and specific.
  • Provide enough context.
  • Set constraints and rules.
  • Choose the right format.
  • Iterate and improve your prompt.
★ A good prompt = clear instructions + relevant context + desired output BETTER PROMPTS → BETTER RESULTS
